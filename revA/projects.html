Projects


Here is a collection of some of my current projects. To see the latest of what I'm workin gon Head over to my github page, or to the Hacks/Hackers Rochester github page
  • BarkingOwl
    BarkingOwl is a scalable web crawler intended to be used to find specific document types on websites, such as PDFs, XLS, TXT, HTML, etc.

    Source Code
  • Monroe Minutes
    Monroe Minutes is a town meeting minutes aggregator for Monroe County, NY.

    Source Code
  • Monroe County, NY 911 Feed Collator
    Monroe County, NY provides an RSS feed of real-time non-violent incidents. The Monroe County, NY 911 Feed Collator website captures that data feed every minute and presents a history of events in the county.

    Source Code
  • LandSAT8 Python Tools
    A small collection of python scripts for working with LandSAT8 data and the ImageMagik, and image processing library and ffmpeg (imgconv) video processing library.

    Source Code
  • Traffairious
    A website that helps display information about the proximity of schools to high traffic roadways in Monroe County, NY.

    Source Code
  • COMIDA Minutes Indexer
    The County of Monroe Industrial Development Agency Meeting Minutes Indexer provides a simple search interface for all COMIDA meeting minutes.

    Source Code
  • Monroe County, NY Jail Census Tools
    Monroe County, NY posts a daily Jail Census in PDF form that is not easily human or machine readable. These tools provide a means to pull down the daily PDF document and convert it into a json file that can be pushed to a nosql database.

    Source Code
